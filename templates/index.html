Instructions

    This is a demo. Presets will not save after app is closed.

  • 1. Set grid size (Rows: y Cols: x) Click set grid.
  • 2. Import folder of pngs. Best practice is to name pngs tile_1.png, tile_2.png etc
  • 3. Use the opacity slider to toggle number/design visibility.
  • 4. Drag tiles (or lock them to freeze). Use + - toggles to zoom. Pinch to zoom on touch devices.
  • ---To swap tile positions make sure both tiles are unlocked.
  • ---Drag the tile you wish to move onto the tile you wish to replace and hold for 2s.
  • 5. Save/Load presets for layouts. You will have to import tiles first before seeing the preset load.
  • 6. Export layout as CSV. You can use this to arrange your carpet
